## 🚀 QUICK START (5 MINUTES)

1. Get COHERE_API_KEY from https://cohere.ai
2. Get GEMINI_API_KEY from https://makersuite.google.com/app/apikey
3. Create backend/.env with your keys (see ENV_EXACT_CONFIGURATION.md)
4. Run: `python -m uvicorn backend.main:app --reload`
5. Test: POST to `/api/ai/chat`
